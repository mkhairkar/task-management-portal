# 📌 Task Management Portal

A full-stack Task Management application built using **React, Node.js, Express, and PostgreSQL**.  
It supports full CRUD operations with filtering, pagination, and validation.

---

## 🚀 Features

- Create tasks
- View all tasks
- Update tasks
- Delete tasks
- Search tasks by title
- Filter by status and priority
- Pagination support
- Form validation (frontend)
- RESTful API integration

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- Vite

### Backend
- Node.js
- Express.js
- PostgreSQL
- dotenv
- nodemon
